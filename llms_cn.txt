# R_LIT

- 用途: Rust 独立工具与库的集合仓库，无根 Cargo workspace
- 主站: https://www.snappywatt.com/
- 预编译包: 托管于 https://www.snappywatt.com/
- 仓库: https://github.com/dwbmio/r_lit

## 目录

- bulk_upload/ — 从 JSON 批量提取 URL 并上传至 S3 兼容存储
- img_resize/ — 图片缩放与压缩
- group_vibe_workbench/ — 桌面协作工作台 (GPUI + P2P)
- pgpour/ — Postgres CDC → Kafka 数据管道（多 topic 扇出，按表路由）
- omniplan_covers_ding/ — 内部工具 (有外部路径依赖，不可移植)
- video-generator/ — 视频工具集
- crates/murmur/ — P2P 协作库 (iroh-net + Automerge CRDT)

## 构建

```bash
cd <tool_dir> && cargo build --release
```

各工具详情见对应子目录内的 README 和 llms.txt。
